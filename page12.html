Game Boy Color:

1998

O Game Boy Color é uma evolução a parte do “Game Boy” original, possuindo uma tela por sua vez totalmente colorida, e também, total compatibilidade com os jogos de seu antecessor, o “Game Boy”.

Acessórios

O Game Boy Color é compatível com quase todos os acessórios do Game Boy, além de incluir o Transfer Pak.
Transfer Pak: Como o nome sugere, o Transfer Pak (Cartucho de Transferência) permite que dados, pontuações máximas, personagens e outros dados sejam transferidos entre os Cartuchos de Jogo para Game Boy Color e os Cartuchos de Jogo para Nintendo 64.

Jogos

Plataforma, puzzles, tiro, estratégia – todos os gêneros estão presentes na vasta biblioteca de mini clássicos da Game Boy. Os jogos de Game Boy Color e os cartuchos originais são compatíveis com a Game Boy Color, oferecendo mais de 800 títulos, incluindo a famosa coleção de Pokémon que conquistou o mundo.
