Game Boy

1989/1996/1997

O Game Boy foi o primeiro console portátil da Nintendo, foi lançado em 1989 e possuía uma tela totalmente monocromática, em 1996 foi lançado uma segunda versão do mesmo console, chamado “Game Boy Pocket”, sendo menor em questão de tamanho e mais viável para ser transportado. Por fim, o Game Boy teve uma última versão lançada em 1997, chamada de “Game Boy Light”, na qual sua nova adição era uma luz interna, para ser possível sua utilização no escuro. 

Acessórios

Cabo Link Game Boy: Conecte-se com outros proprietários de Game Boy usando o Cabo Link. O cabo não funciona com a Game Boy Advance. Com o Cabo Universal Game Link, é possível conectar duas Game Boy Pocket/Game Boy Color ou uma Game Boy original e uma Game Boy Pocket sem adaptadores. Verifique o ícone do Cabo Link nos cartuchos compatíveis.
Super Game Boy: Revitalize os Cartuchos de Jogo clássicos de Game Boy com a Super Game Boy, que projeta jogos para a tela grande sem precisar de uma consola Game Boy. Ela transforma jogos em preto e branco em versões coloridas, com até 256 cores, e permite criar molduras personalizadas. Com a Super Game Boy 2, você pode conectar a Super Nintendo e outro Game Boy para mais diversão. Note que não é compatível com cartuchos de Game Boy Color ou Game Boy Advance.
Game Boy Camera: Com um encaixe perfeito na ranhura para jogos da Game Boy, este acessório inteligente permite capturar imagens em preto e branco com um toque de botão. Em seguida, você pode colar, cortar, unir e até adicionar personagens da Nintendo às fotos usando as ferramentas integradas.

Jogos

Todas as tuas personagens Nintendo favoritas – Mario, Yoshi, Pokémon, Donkey Kong, Samus Aran, etc. – podem ser transportadas para onde quer que te desloques, graças ao catálogo de mais de 450 jogos da Game Boy. A Game Boy tem tudo, quer procures jogos de tiro, jogos de plataforma, puzzles, jogos de luta ou até títulos desportivos.
